**TEMPLATE DE TRACKER DE METAS E OKRs**

| **Parâmetro** | **Descrição** |
| --- | --- |
| Objetivo Trimestral | Define os objetivos a serem alcançados durante o período de 3 meses |
| Mês/Ano | Ano/Tempo |
| Responsável | Identifique quem será responsável pela execução do objetivo |
| Prazo | Defina o prazo de entrega ou cumprimento do objetivo |

**Tabela de Metas Trimestrais**

| **Parâmetro** | **Descrição** |
| --- | --- |
| Título da Meta | Objetivo a ser alcançado durante o período de 3 meses |
| Data Início | Data inicial para iniciar a execução da meta |
| Prazo | Defina o prazo para completar a meta |
| Objetivos Submetidos | Identifique os objetivos específicos que serão alcançados ao longo do trimestre |
| Tarefa 1 | Descrição detalhada da tarefa ou atividade necessária para alcançar o objetivo |
| Resultado | Defina como será medido a progresso e o resultado final da tarefa |

**Respostas Pessoais**

| **Parâmetro** | **Descrição** |
| --- | --- |
| Nome | Descreva o nome do indivíduo responsável pela execução do objetivo |
| Metas Trimestrais | Liste as metas trimestrais específicas de cada pessoa |
| Tarefas | Descreva as tarefas e atividades que cada pessoa deve realizar para alcançar suas metas |

**Respostas Profissionais**

| **Parâmetro** | **Descrição** |
| --- | --- |
| Nome | Descreva o nome do indivíduo responsável pela execução do objetivo |
| Metas Trimestrais | Liste as metas trimestrais específicas de cada pessoa |
| Tarefas | Descreva as tarefas e atividades que cada pessoa deve realizar para alcançar suas metas |

**Revisão Mensual**

| **Parâmetro** | **Descrição** |
| --- | --- |
| Data de Revisão | Defina a data em que as metas trimestrais serão revisadas |
| Objetivos Trimestrais | Liste os objetivos trimestrais e suas respectivas metas |
| Progresso | Descreva o progresso alcançado até a data de revisão |

**Exemplo de Revisão Mensal**

| **Parâmetro** | **Descrição** |
| --- | --- |
| Título da Meta | Definir os objetivos para o próximo trimestre |
| Data Início | 1º de março |
| Prazo | 30 de junho |
| Objetivos Submetidos | Alcançar a meta de aumentar a produtividade por 20% durante o trimestre |
| Tarefa 1 | Definir as tarefas necessárias para alcançar a meta, incluindo treinamento e atualização da equipe |

Essa é uma estrutura básica para criar um template de Tracker de Metas e OKRs. Você pode personalizá-la conforme necessário e ajustar as tabelas e campos de acordo com suas necessidades específicas.
